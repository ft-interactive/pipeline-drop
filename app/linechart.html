extended pipeline options

  • &height={total chart height}
  • &width={total chart width}
  • &chartHeight={height of the plot area}
  • &chartWidth={width of the plot area}
  • &blockPadding={vertical space between chart elements}
  • &indexProperty={name of the column with the dates in (defaults to '&') }
  • &falseorigin={true or false, if it's true then the chart won't start at 0}
  • &simpleDate={true or false, if true then the date axis will only have first and last values and will be trimmed as close as possible to the date range}
  • &simpleValue={true or false, if true then the value axis will have first last and (if appopriate) 0 and will be trimmed as close as possible to the value range]
